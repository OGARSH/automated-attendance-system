# Logo Instructions

To add your logo.png file:

1. Create or find your logo image (PNG format recommended)
2. Name it "logo.png" 
3. Place it in the root directory of your project (same folder as index.html)
4. The logo should ideally be:
   - 32x32 pixels for favicon
   - 192x192 pixels for web app icon
   - Square format works best
   - PNG format with transparent background

The index.html is already configured to use logo.png as:
- Favicon (shows in browser tab)
- Web app icon

If you don't have a logo yet, you can:
- Use an online logo maker
- Create one using any image editor
- Use a simple icon that represents your attendance system
